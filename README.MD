# COFFEE FRAMEWORK (CSS)

## Grid System
A 12 column layout system. this is on top of the 8pt grid system.

* Preprocessor - [SCSS](https://sass-lang.com/)
* Build System - [gulp.js](https://gulpjs.com/)
